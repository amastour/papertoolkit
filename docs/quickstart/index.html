Quick Start

  1. Get an Anoto Pen. We use the Nokia SU-1B, but the Maxell Penit probably also works.
  2. Install the Nokia SU-1B software and the Anoto SDK.
  3. If the pen hasn't been upgraded, upgrade the firmware to allow for streaming. Install the Widcomm drivers for your bluetooth dongle.
  4. Make sure you have .NET 2.0.
  5. For handwriting recognition, install the Tablet PC SDK ver 1.7 and the Recognizer Pack. This only works on Windows Pro/Tablet editions. If you have a Mac, unfortunately, you'll need to dual boot or run Parallels.
  6. Eclipse IDE
    Install this IDE, because the R3 toolkit is primarily in Java. If you need to customize any of the .NET parts of R3 (which is unlikely) you'll need Visual Studio .NET 2005.
  7. Java 6
    Install the JDK/JRE.
  8. TortoiseSVN
    Get Tortoise v. 1.4 so that you can download the code.
  9. Download the code from Google Code Hosting. Use the HTTPS version if you are a developer on this project. Otherwise, use the HTTP version (which is displayed below).
    http://papertoolkit.googlecode.com/svn/trunk/
    This is the main PaperToolkit project. I tend to put it under My Documents/Projects/PaperToolkit, but you can pick your favorite location.

    and

    http://papertoolkit.googlecode.com/svn/demos/trunk/
    This contains the demonstration apps. I usually put it under My Documents/Projects/PaperToolkitDemos.
  10. Java Comm
    Under PaperToolkit\lib\comm, copy the comm.jar and javax.comm.properties files into the right places
    C:\Program Files\Java\jdk1.6.0\jre\lib
    and
    C:\Program Files\Java\jre1.6.0\lib
  11. Install SVNKit (formerly known as JavaSVN) plugin for Eclipse.
  12. Install Subclipse or Subversive. They both seem to have plusses and minuses, although I currently have Subversive installed. Make sure you have subclipse use the SVNKit or JavaSVN that supports SVN 1.4, otherwise tortoise won't work well with your Eclipse SVN plugins.
  13. The first demo you can try is the Handwriting Recognition demo....
