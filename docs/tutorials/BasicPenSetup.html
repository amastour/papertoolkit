Basic Pen Setup

Nokia SU-1B

  1. If you bought an SU-1B, you should have a CD that Nokia provided. Pop in the CD, and follow the instructions.
  2. Once you have installed & rebooted, you should be able to write on a commercial Anoto notebook (e.g., Esselte A5, Mead 5x8 Cambridge Limited) and drop your pen into the cradle. You will see a dialog with an upload progress bar. Once it is done, you can open your digital notes using the Nokia-provided browser, and do fun stuff like animate your handwriting.
  3. The default software allows you to copy ink strokes to external applications, such as MS Word.
