The R3 API

This is where you can start exploring the R3 API. While you can feel free to dive in and use the individual packages, it is easier to use these packaged methods until you are more familiar with the toolkit.

Augmented Sheet
The AugmentedSheet class represents a sheet of paper that is augmented with interactive capabilities. Add child objects to this class to build up a sheet that includes images, shapes, and dot pattern. The sheet can later be printed to PostScript, PDF, or straight to a Java2D printer (with different advantages to each printing technique, unfortunately).

Augmented Document (AugmentedDocument.java)
You can bundle a set of AugmentedSheets into a document. If you want to prototype a notebook interaction (like in ButterflyNet), you would want to create an AugmentedDocument.
